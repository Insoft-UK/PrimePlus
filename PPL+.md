# PPL+

## if...then...else...endif
```
if A >= B then
  statement/s
else
  statement/s
endif; // or PPL 'END;'
```

## switch
```
switch A // PPL 'CASE'
  case 1 do // PPL 'IF A==1 THEN'
  end;
default
end;
```

## while...wend
```
while A <> 0
wend; // or PPL 'END;'
```

## try...catch
```
try
  statement/s
catch
  statement/s
end;
```
